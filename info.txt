üîê API ENDPOINT'LAR:

1. AUTHENTICATION:
   POST /register - Foydalanuvchi ro'yxatdan o'tish
   POST /login - Tizimga kirish

2. GROUPS:
   POST /groups - Yangi guruh yaratish (Matematika, Fizika...)

3. QUESTIONS:
   POST /questions - Yangi savol yaratish (4 javob variant)
   GET /questions/all - Barcha savollarni olish
   GET /questions/test - Test uchun savollarni olish

4. USERS:
   GET /users - Barcha foydalanuvchilarni olish
   DELETE /users/{username} - Foydalanuvchini o'chirish
   DELETE /users - Barcha foydalanuvchilarni o'chirish

5. SYSTEM:
   GET /health - Server holatini tekshirish

üåê URL: https://it-zone.uz/

üìã CURL KOMANDALARI:

REGISTER:
curl -X POST "https://it-zone.uz/register" \
  -H "Content-Type: application/x-www-form-urlencoded" \
  -d "username=teacher1&password=123456"

LOGIN:
curl -X POST "https://it-zone.uz/login" \
  -H "Content-Type: application/x-www-form-urlencoded" \
  -d "username=teacher1&password=123456"

CREATE GROUP:
curl -X POST "https://it-zone.uz/groups" \
  -H "Content-Type: application/x-www-form-urlencoded" \
  -d "token=YOUR_TOKEN&title=Matematika"

CREATE QUESTION:
curl -X POST "https://it-zone.uz/questions" \
  -H "Content-Type: multipart/form-data" \
  -F "token=YOUR_TOKEN" \
  -F "group_title=Matematika" \
  -F "text=2+2=?" \
  -F "answer1=3" \
  -F "answer2=4" \
  -F "answer3=5" \
  -F "answer4=6" \
  -F "correct_answer=2"

GET QUESTIONS:
curl -X GET "https://it-zone.uz/questions/all" \
  -H "Authorization: Bearer YOUR_TOKEN"

GET TEST:
curl -X GET "https://it-zone.uz/questions/test?token=YOUR_TOKEN&group_title=Matematika"

üéì QANDAY ISHLAYDI:
